Factores socioeconomicos en el consumo del alcohol : 2023¶

Tarea 1¶

Este estudio se centra en analizar los factores socioeconómicos que influyen en el gasto en bebidas alcohólicas en el Perú entre los años 2019 y 2023, utilizando datos de la Encuesta Nacional de Hogares (ENAHO). Para ello, se integran los módulos de Características de la vivienda y hogar (Módulo 1) y Gasto en alimentos y bebidas (Módulo 7).

import pandas as pd

# Definir el tamaño de los chunks (bloques)
chunksize = 100000  # Ajusta este valor según la memoria disponible

# Lista para almacenar los datos filtrados
filtered_chunks = []

# Lista de palabras clave para bebidas alcohólicas
bebidas_alcoholicas = ['Whisky', 'Ron', 'Pisco', 'Cerveza', 'Vino', 'Otros']

# Leer y procesar por chunks
for chunk in pd.read_csv(ruta_modulo_7, encoding="latin1", chunksize=chunksize, usecols=['UBIGEO', 'P601X', 'P601B', 'P601C', 'P601B1']):
    # Filtrar productos relacionados con bebidas alcohólicas
    chunk['Es_Alcohol'] = chunk['P601X'].str.contains('|'.join(bebidas_alcoholicas), case=False, na=False)
    filtered_chunk = chunk[chunk['Es_Alcohol']]  # Filtrar solo bebidas alcohólicas
    filtered_chunks.append(filtered_chunk)  # Almacenar el chunk filtrado

# Combinar todos los chunks filtrados
modulo_7_alcohol = pd.concat(filtered_chunks, ignore_index=True)

# Mapear la frecuencia de compra (P601B1)

    1: "Diario",
    2: "Interdiario",
    3: "Semanal",
    4: "Quincenal",
    5: "Mensual",

    8: "Semestral",
    9: "2 veces por semana",

import numpy as np
import scipy.stats as stats

# Filtrar los datos para la categoría "Casa independiente"
casa_independiente = data_combinada[data_combinada['Tipo_Vivienda'] == 'Casa independiente']['log_gasto_bebidas']

# Calcular el promedio general
promedio_general = data_combinada['log_gasto_bebidas'].mean()

# Estadísticas descriptivas de "Casa independiente"
media_casa = casa_independiente.mean()
desv_est_casa = casa_independiente.std()
n_casa = len(casa_independiente)

print("Estadísticas de 'Casa independiente':")
print(f"Media: {media_casa:.2f}, Desviación Estándar: {desv_est_casa:.2f}, N: {n_casa}")

# Prueba de hipótesis (t-test para una muestra)
t_stat, p_value = stats.ttest_1samp(casa_independiente, promedio_general)
print("\nPrueba de Hipótesis:")
print(f"Estadístico t: {t_stat:.3f}, Valor-p: {p_value:.3f}")

# Interpretación de la prueba
alpha = 0.05
if p_value < alpha:
    print(f"\nRechazamos H₀: La media del gasto en 'Casa independiente' es significativamente diferente del promedio general (α={alpha}).")
else:
    print(f"\nNo se puede rechazar H₀: No hay evidencia suficiente para afirmar que la media del gasto en 'Casa independiente' es diferente del promedio general (α={alpha}).")

# Calcular intervalo de confianza del 95% para "Casa independiente"
confianza = 0.95
error_estandar = desv_est_casa / np.sqrt(n_casa)
ic_min, ic_max = stats.t.interval(confianza, df=n_casa-1, loc=media_casa, scale=error_estandar)

print("\nIntervalo de Confianza del 95% para la media de 'Casa independiente':")
print(f"[{ic_min:.2f}, {ic_max:.2f}]")
